공합니다. "numbers" 모듈은 숫자 데이터 타입을 위한 추상 계층 구조를 정

함수를 가지고 있습니다. "decimal" 모듈은 임의의 정밀도 계산을 사용하여
정확한 10진수 표현을 지원합니다.
